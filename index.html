AI Learning Tool Adoption and Recommendation System

A web-based educational platform designed to improve awareness, responsible usage, and adoption of AI learning tools among students.

This system includes AI literacy modules, AI tool recommendations, self-assessment activities, prompt practice, and prescriptive analytics-based recommendations.

System Features

  1. AI Tool Awareness Modules
  2. AI Tool Recommendation Engine
  3. AI Literacy Self-Test
  4. Prompt Practice and Feedback
  5. Responsible AI Usage Guidance

Developed using ADDIE + Waterfall SDLC + TAM framework integration.
