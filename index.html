AI Learning Tool Adoption and Recommendation System

A web-based educational platform designed to improve awareness, responsible usage, and adoption of AI learning tools among students.

The system includes AI literacy modules, Pre-Test and Post-Test Likert-scale surveys, AI tool recommendations, self-assessment, and prompt practice.

System Flow

  1. Answer the Pre-Test Survey
  2. Explore AI Learning Tools
  3. Use the AI Recommender
  4. Practice Prompt Writing
  5. Answer the Post-Test Survey

Developed using ADDIE, Waterfall SDLC, and TAM integration.
